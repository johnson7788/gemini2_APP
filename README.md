## Installation Steps

Clone the project
```
git clone <repository-url>
cd geminicall
```

Install dependencies

```
npm install
```

Configure environment variables

```
cp env_template .env
```

Start the development server

```
npm start
```

Build the production version

```
npm run build
```

Run the production version

```
npm run start
```
